# Instructions

Complete the function CrossProduct that takes two tuples x and y of length at most 3 as input and returns their cross product.

Reference: https://en.wikipedia.org/wiki/Cross_product#Computing

# Examples

Input

>1 2 3
>
>4 5 6

Output

>(-3, 6, -3)

Input

>1 2
>
>4 5 6

Output

>(12, -6, -3)

Input

>1 2
>
>4 5

Output

>(0, 0, -3)
